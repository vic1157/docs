## Directory Overview

```
src/
├── pages/               # Route components (24 pages)
├── components/          # Reusable components (8 custom + 46 UI)
├── hooks/               # Custom React hooks (3 hooks)
├── lib/                 # Utility functions
└── App.tsx              # Main routing configuration
```
