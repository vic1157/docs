## Activity Panel

Interactive bottom sheet panel for accessing AI features while reading.

<Tabs>
  <Tab title="Inactive Mode">
    **Reference Reading** (not tracked)

    **Contracted:**
    - Minimal view with drag handle

    **Extended:**
    - Start Scroll button
    - Chapter Context button
    - Ask Leo button
  </Tab>

  <Tab title="Active Mode">
    **Tracked Reading** (Scroll in progress)

    **Contracted:**
    - Reading timer display

    **Extended:**
    - Chapter Context button
    - Add Question button
    - Ask AI button
    - Reading timer
    - Bookmark button
  </Tab>
</Tabs>

## Panel Features

<AccordionGroup>
  <Accordion title="Drag Gestures" icon="hand">
    - Powered by Framer Motion
    - Smooth spring animations
    - 50px drag threshold
    - Tap handle to toggle states
    - Swipe down to collapse

  <Accordion title="State Management" icon="toggle-on">

    - Expanded/Contracted panel states
    - Timer for active sessions
    - Visual feedback with borders and separators
  </Accordion>

  <Accordion title="Z-Index Hierarchy" icon="layer-group">
    - Panel: z-40
    - Footer navigation: z-50
    - Ensures navigation always accessible
  </Accordion>
</AccordionGroup>

## "Start Scroll" Flow

When user clicks "Start Scroll":
1. Panel transitions to Active mode
2. Reading timer begins
3. AI action buttons appear
4. User can add questions and reflections
5. Session tracked until completion
