## Chat Interface

Available at `/chat` with support for existing chats via `/chat?id={chatId}`.

<Tabs>
  <Tab title="Features">
    - Message history display (user and Leo messages)
    - Real-time text input with send button
    - Auto-scroll to latest message
    - Enter key to send messages
    - Clean left/right aligned message bubbles
  </Tab>

  <Tab title="Mobile Optimization">
    - Fixed layout (`fixed inset-0`) for true full-screen
    - Proper keyboard handling
    - Safe-area padding: `env(safe-area-inset-bottom)`
    - 16px minimum font size to prevent iOS auto-zoom
    - Input anchored to bottom regardless of keyboard state
  </Tab>
</Tabs>

## Technical Details

### ChatMessage Component

```typescript
interface ChatMessage {
  id: string;
  sender: 'user' | 'leo';
  content: string;
  timestamp: string;
}
```

Message bubbles styled based on sender:
- **User messages**: Right-aligned
- **Leo messages**: Left-aligned with AI branding

### Chat Interface

```typescript
interface Chat {
  id: string;
  title: string;
  messages: ChatMessage[];
  createdAt: string;
  updatedAt: string;
}
```
