## Overview

Scripture Explorer serves as the central hub for accessing all scripture interaction features. Accessible via the middle "Create" button in the Footer navigation.

## Main Features

<Tabs>
  <Tab title="Action Buttons">
    Two prominent Card-based action buttons:
    - **New Chat**: Starts fresh conversation with Leo
    - **New Quiz**: Navigate to quiz creation page
  </Tab>

  <Tab title="Tabbed Content">
    Toggle between two views:
    - **Scrolls Tab**: List of all scrolls with quiz status
    - **Chats Tab**: Conversation history with last messages
  </Tab>
</Tabs>

## Navigation Flow

```
Footer "Create" Button → /scripture-explorer
    ├── New Chat → /chat (new conversation)
    ├── New Quiz → /new-quiz → select scroll → /scroll/:id?startQuiz=true
    ├── All Scrolls → click scroll → /scroll/:id
    └── All Chats → click chat → /chat?id={chatId}
```
